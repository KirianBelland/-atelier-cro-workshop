📌 Organisation des Insights

Organisez vos insights par dossiers thématiques

💡 Comment utiliser les Insights Clés ?

  • Titre clair : Formulez un insight concis et actionnable
  • Description : Ajoutez du contexte et des chiffres clés
  • Priorité : Triez vos insights par importance (Haute/Moyenne/Basse)
  • Lien de redirection : Connectez l'insight à une section spécifique de l'atelier
  • Navigation intelligente : Le bouton vous amène directement à l'élément avec highlight automatique
Exemple d'insight :

"Page Resort - Trafic élevé mais conversion faible"
→ 68% du trafic passe par cette page mais seulement 12% entrent dans le BE
→ Lien vers : Parcours Utilisateurs > "Resort FR"
